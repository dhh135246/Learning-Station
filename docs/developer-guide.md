## 一、 PWA 桌面独立 App 安装与数据同步指南 (PWAインストール・データ同期)

本项目现已全面升级支持 **PWA (Progressive Web App，渐进式 Web 应用)** 规范。您可以一键将其“安装”到您的电脑桌面，生成无边框的本地独立 App 窗口，解除传统浏览器的标签页 and 插件干扰，并大幅提升 Canvas 落樱与 Web Audio 音频发声性能。

### 1. 简易安装步骤 (Chrome / Edge 浏览器)
1. **启动本地服务器**：在项目根文件夹下运行以下命令启动轻量级静态服务：
   ```bash
   npx serve -l 8080
   # 或者使用 Python：python -m http.server 8080
   ```
2. **访问应用**：打开 Chrome 或 Edge 浏览器，访问 `http://localhost:8080`。
3. **一键安装**：
   * **PC 端**：观察浏览器地址栏右侧，会浮现一个“安装”或“添加桌面快捷方式”的屏幕小图标。点击并确认安装。
   * 系统将自动在桌面生成独立的应用图标，并弹出无浏览器地址栏的 **独立 Standalone 运行窗口**。

### 2. 双击直接运行 (`file:///` 协议) 与 PWA 本地服务器版的数据同步边界
由于浏览器内核的数据沙盒同源安全策略：
* 直接在文件管理器中双击打开的 `file:///f:/Japanese/customised_system/index.html` (Origin: `null`) ；
* 与通过本地服务器运行并安装的 `http://localhost:8080` (Origin: `http://localhost:8080`)。
两者的 `LocalStorage` 属于不同的安全沙盒，数据**默认不互通**。

**数据同步闭环方案**：
我们为此重构了极其健壮的 **「导出备份 ➔ 拖拽还原」数据闭环**。如果您在本地双击运行时积累了学习进度，只需在控制中心点击“导出备份数据 (JSON)”，然后在 PWA 桌面 App 窗口的“控制中心”将该 `.json` 文件拖入📁拖放盘，系统即会秒级反演并完美同步您的所有假名掌握进度、SRS生词记忆周期以及私人语法笔记库！

## 二、 项目定位与设计美学 (プロジェクト概要)

*   **目标用户**：中文母语的日语初学者，拥有 50 音基础但不够熟练，看动漫较多，无系统语法背景，偏好 PC 端键盘录入。
*   **设计美学：极致日式和风 (Wafuu Modern style)**
    *   **色调系统**：采用以和纸白 (`#FDFBF7`)、朱砂红 (`#B83E38`)、竹席黄 (`#F4EFE3`)、抹茶绿 (`#4A6B53`) 为核心的高质感纸质色调。
    *   **暗色和纸夜间模式（夜樱版）**：基于“消炭色” (#19181F) 背景、“桔梗紫” (#22212B) 卡片、配合“发光樱粉/流光金”的前景色体系。支持本地持久化存储，且切换时伴有清脆的敲竹声学反馈。
    *   **字型优化**：实施 **中日文字型（Font Stack）隔离机制**。全局 `body` 采用简体中文系统字体栈，消除汉字显示扭曲；日语核心区包裹 `.jp-text` 强制调用日文字体栈（如 Sawarabi, Yu Gothic, Meiryo, MS Mincho），保障离线断网时字形完美。

## 四、 核心代码逻辑与模块详解 (七大主要功能 Tab 业务)

项目采用 **“解耦的组件式单页面”** 形式开发。每个主菜单 Tab 在 `index.html` 中都有其对应的独立 `<section id="[tab-id]" class="tab-content">`。以下是核心的七大 Tab 板块具体业务和底层算法流程：

### 1. 🏠 学习看板 (`id="dashboard"`)
*   **打卡签到**：每日跨天首次登录系统时触发，播放竹木清脆击石声，累计连续学习天数 `streak`，持久化至 `localStorage`。
*   **数据大统计卡**：实时读取单词本库中的“总单词数”、“已掌握单词”以及基于 SRS 遗忘时间戳过滤得出的“待复习单词”数目。
*   **未来 7 天待复习单词预测日历 (Forecast Calendar)**：
    *   遍历词库中每个单词的 SM-2 记忆到期时间，预测未来 7 天内每天将要到期的单词数量。
    *   根据待复习数对格子进行色块化着色：`轻松` (灰色/暖黄, 0个)、`少量` (浅绿, 1-3个)、`适中` (深绿, 4-8个)、`较多` (浅红, 9-15个)、`高峰` (深红, >15个)。其“今天”格子所显示的数值与上方看板的待复习单词数保持强一致性。

### 2. 🌸 50音庭園 (`id="kana"`)
*   **错落滑入**：假名卡片在渲染挂载时，会触发扇形旋转轨迹错落滑入（Sensu Unfold）动画，增强视觉张力。
*   **发音系统**：基于原生 `window.speechSynthesis`。核心逻辑封装 in `AudioCenter` 中，自适应匹配 `ja-JP` 日语原生语音。
*   **假名测验与错题集**：支持选择范围（清音、浊音、拗音）和题型，错题更新在 `kana_mastery` 计数中，并在控制中心动态渲染为薄弱点看板。
*   **易混淆专项**：预设了五组高度相似的假名（如：`ね/れ/わ`，`さ/ち`），帮助快速脱盲。
*   **手写临摹沙盒**：配合和纸背景，利用 HTML5 Canvas 绘制以朱砂红（`#B83E38`）为笔触颜色的书写墨迹，支持一键清空重写。

### 3. 🛠️ 語法工房 (`id="grammar"`)
*   **系统化自学教程**：大纲设计了 9 章共 31 节的基础日语语法，详细的数据结构存储在外部解耦文件 `js/grammar-course-data.js` 中，由 `index.html` 离线极速渲染阅读，无任何跳转延迟。
*   **核心句型卡**：收录了 68 项核心分级语法点，支持根据 N5~N1 进行子标签过滤。卡片点击可展开模态框查看具体接续，并包含配备 TTS 点击发音的多条实用日汉例句。
*   **私人语法笔记导入与本地渲染**：
    *   支持在控制中心拖入本地 `.md` 文件或复制 Markdown 源码同步保存至本地 `localStorage` 的 `custom_grammar_notes` 中。
    *   内置本地 RegExp 渲染器，支持 `#` / `##` / `###` 标题、`**` 粗斜体、GitHub alert 提示框及带 `🔊` 点击朗读发音链接（`play://` 协议）。
    *   私人笔记卡片带有专属红色左框线 `.private-note-card` 以示高亮区分，支持“📓 私人笔记”一键过滤。

### 4. 📖 単語帳 (`id="vocab"`)
*   **卡片翻转与 SRS 自测**：支持中日/日中双向记忆，复习卡牌支持 180 度中轴立体物理旋转 (rotateY) 动画。内置 SM-2 算法，根据打分更新容易度因子 `easeFactor`、下次复习间隔天数 `interval` 和连续正确复习次数 `repetitions`，并计算 `nextReviewDate`。
*   **智能排序**：支持按 `createdAt` 降序/升序或记忆到期时间进行智能排序，确保生词添加、删除与恢复时顺序合理。
*   **字形隔离与分页**：列表仅对日语词汇和假名加载 `jp-text` 字体，中文释义则渲染为系统默认中文，防止异形字错乱。配合精美分页控件。

### 5. 🎮 次元道场 (`id="parser"`)
趣味学习核心 Tab，包含四个子板块：
*   **文本解析轨 (Lyrics Parser)**：
    *   **打字输入**：支持自定义歌词/台词打字挑战。输入框自动挂载罗马字假名实时转换转换器，在完美匹配时框边呈绿色，并在输入行上方实时展现匹配进度百分比。打完通关后触发 Web Audio 古琴胜利琶音与 Canvas 落樱驟雨大秀。
    *   **Trie 树智能分词与碰撞**：基于前缀 Trie 树匹配提取汉字与外来语词组。提取出的生词自动与系统词库进行双重碰撞，已匹配的单词自动预填读音与释义并高亮亮起绿色的 `已匹配` 徽章，支持一键同步。
    *   **查词**：分词生成的卡片右侧挂载 `🔍` Weblio中日查词按钮。
*   **台词大乱斗 (Quotes Matcher)**：动漫名台词连连看配对卡牌游戏，包含 TTS 语音发音与翻转动画。
*   **体育热词卡 (Sports Glossary)**：足球与棒球的高频外来语（片假名 loanwords）配对挑战。
*   **异世界冒险 (Isekai RPG Quest)**：
    *   **波次序列**：3只普通怪 ➔ 1只精英领主的固定战斗序列循环。普通怪积 1 分，领主积 2 分，积满 10 分第 9 场刷新关底大 Boss 魔王赤鬼 `👹` (500HP)。
    *   **药水机制**：无伤完美通关领主可获取「生命药水 🧪」，使用瞬间恢复 50% HP（耗费 1 回合步骤）。
    *   **免 CORS 查词**：遭遇战/战斗字词汇旁挂载 `🔍` 按钮，直接弹窗查词。
    *   **职业称号与英雄榜**：备战区根据所选词库匹配称号，通关魔王后按耗费最少步骤取前十进入 local 排行榜。

### 6. ⚙️ 控制中心 (`id="data"`)
*   **数据备份与拖拽还原**：
    *   **Blob 强健导出**：导出时采用 Blob 二进制流和 `URL.createObjectURL` 替代 base64 编码，打包私人语法笔记并自动触发内存释放机制，防止内存泄漏或浏览器溢出崩溃。
    *   **和纸拖拽文件还原盘**：数据管理区域重构为带有📁标志的 `.washi-drop-zone` 拖放盘，支持直接拖入备份 JSON 文件自动解析还原进度。
*   **私人语法笔记导入区**：支持拖入 Markdown 文件或在此手动粘贴 Markdown 源码同步保存私人笔记。
*   **薄弱假名看板**：提取错题集的假名，以醒目红字展示薄弱假名。

### 7. 🌐 资源导航 (`id="resources"`)
*   收录了 **日语语法指南 (Tae Kim's Guide)**、**OJAD (在线日语声调辞典)** 等精选辅助网站，为学习者提供快捷的外部辅助学习入口。

## 五、 额外功能介绍 (追加機能について)

不严格归属于某一个 Tab 内的独立添加的子功能（如输入转换、查词工具、声效与粒子背景等）均在此做底层技术梳理：

### 1. 「罗马字 ➔ 假名」输入法实时转换引擎 (`romaji-converter.js`)
*   **三模键盘快捷键切换**：平假名 `あ` ➔ 片假名 `ア` ➔ 英文 `A`。支持点击输入框右侧的徽章或按下快捷键 `Alt + Q` 或 `Ctrl + M` 进行无缝循环切换，切换时播放不同赫兹的 Web Audio 蜂鸣音效。
*   **光标位置防跳跃保护 (Caret Position Preservation)**：在字符串中段插入罗马字修改时，输入法会自动计算当前的绝对 Selection 位置，将替换后的假名插在正确位置，**彻底解决了传统输入法替换假名时导致光标强行跳跃到字符串最末尾的痛点**。
*   **末尾 `n` 自动拨音补全**：打字时单个 `n` 保持不变，当按下回车提交或输入框失去焦点（Blur）时，末尾残留的 `n` 自动补全为拨音 `ん` 或 `ン`。
*   **自适应样式克隆系统**：通过 `RomajiConverter.bind()` 绑定时，系统会自动抓取原文本框的布局属性（如 `flex-grow`、`margin`、`display` 等）并全额赋予外包围器，彻底解决了因加入切换徽章挤压周围 Flex 布局导致的换行和变形痛点。

### 2. Trie 树智能双语分词与碰撞模块 (`japanese-tokenizer.js`)
*   基于 Trie 前缀树匹配构建的轻量级分词引擎，自动提取汉字和片假名外来语词组。
*   提取出的生词自动与系统词库进行双重碰撞，若匹配则自动预填该词的读音假名与中文释义，并标记绿色已匹配徽章。

### 3. 免同源 CORS 突破查词与避让式侧边悬浮词典抽屉
*   **Weblio (中日) 词典源接入**：替换已失效的沪江小D为 `https://cjjc.weblio.jp/content/` 词典源，自动智能匹配日中/中日查词。
*   **侧边折叠面板**：右下角常驻 `🔍` 悬浮圆钮，点击在右侧平滑滑出宽为 `360px` 的“随身辞书”面板。
*   **宽度平滑避让，对着边看边打**：展开时主内容容器 `main` 的宽度会自动由 `calc(100% - 280px)` 平滑缩窄为 `calc(100% - 280px - 360px)`，主界面内容向左偏移，100% 杜绝遮挡打字输入区和主视图。背景无遮罩，不虚化，支持打字后在输入框按 Enter 键一键检索。
*   **快捷查词与随身辞书联动**：点击单词卡片上的 `🔍` 按钮，依然可以在页面中心调起带模糊背景的浮动查词弹窗。

### 4. PWA Service Worker 离线静态缓存拦截 (`sw.js` & `manifest.json`)
*   引入 `sw.js` 进行离线静态资源本地缓存，采用 `Stale-While-Revalidate` 策略。
*   通过 `if (window.location.protocol.startsWith('http'))` 安全锁进行安全沙盒判断，成功规避了双击 `file:///` 协议打开时 Service Worker 注册报错。

### 5. Web Audio API 物理模拟声学系统 (`wafuu-audio.js`)
*   纯本地客户端运行，无需网络。利用原生 `OscillatorNode`（三角波/锯齿波/方波/正弦波）与 `GainNode` 合成和风传统敲竹音效（“笃”声）与古琴胜利琵琶扫弦琶音。

### 6. Canvas 樱花粒子动效与夜樱发光模式 (`sakura-effect.js`)
*   使用原生 Canvas 在背景中融入落樱微动效。在“夜间模式”下，花瓣的透明度和发光模糊效果（Glow Effect）自适应调亮，并提供庆祝大面积樱花雨落花大秀的调用接口 `SakuraEffect.shower(durationMs)`。

### 7. SRS 每日内置单词入队限额与平滑分流机制
*   **配置入口**：在「⚙️ 控制中心」的“备份与控制中心”卡片内，用户可以自行输入并修改每日新词入队额度（默认15个/天，范围 $5 \sim 100$ 个/天），点击“保存并重排”确认生效并播放敲竹音效反馈。
*   **SM-2 算法核心与打分反馈 (0-5分)**：
    *   **容易度因子 (`easeFactor`)**：默认初始为 2.5。每次根据用户反馈微调。高 `easeFactor` 代表单词好记，复习间隔增加；答错则会降低。
    *   **复习间隔天数 (`interval`)**：控制下一次单词到期的天数（如 1天 ➔ 6天 ➔ 14天 等）。
    *   **连续正确次数 (`repetitions`)**：连续选对或标记认识的次数，答错则清零。
    *   **评分标准**：
        *   `❌ 没记住 (0-2分)`：完全没记起。`repetitions` 归零，`interval` 设为 1 天，该词明日自动重新加入复习队列。
        *   `模糊 (3-4分)`：回忆吃力。`repetitions` 加 1，复习间隔依容易度因子相乘温和递增。
        *   `🌸 记牢了 (5分)`：极其熟练。`easeFactor` 上调，复习间隔跨越式拉长（甚至几个月后才会唤醒复习）。
*   **首发天级平滑分流与积压自动重排**：
    *   为了防止 500+ 个默认内置词库在刚启动或升级时“全部扎堆在今天到期”造成学习压力暴载，系统设计了 **「平滑天级分流与积压自动重排」** 机制。
    *   系统会自动检查如果积压的未复习词超过 25 个，自动把它们按设定的日限额为一组，天级平铺排程到未来的 30+ 天中，使主看板的未来 7 天复习日历方块均匀亮起抹茶绿色，保障了抗遗忘排程的科学性与和风视觉的波动美感。
