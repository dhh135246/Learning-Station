# 日本語入力ガイド / 日语键入完整指南

> 🖥️ 平台：Windows 10/11
> ⌨️ 输入方式：罗马字输入法（ローマ字入力）

---

## 一、安装与配置

### 1.1 添加日语输入法

1. 按 `Win + I` 打开「设置」
2. 进入「时间和语言」→「语言和区域」
3. 点击「添加语言」→ 搜索「日本語」→ 安装
4. 安装完成后，任务栏右下角会出现语言切换图标

### 1.2 切换输入法

| 快捷键 | 功能 |
|--------|------|
| `Win + Space` | 在中文/日语输入法之间切换 |
| `Alt + ~`（波浪键） | 在日语输入法内，切换「英数」和「假名」模式 |
| `Alt + Shift` | 备选切换方式 |

### 1.3 输入法模式说明

切换到日语输入法后，任务栏会显示 `あ` 或 `A`：
- **あ（Hiragana）**：日语假名输入模式 ← **主要使用这个**
- **A（Alphanumeric）**：英文直接输入模式

### 2.2 特殊输入技巧

| 输入内容 | 输入方法 | 说明 |
|----------|----------|------|
| 促音っ | 双打下一个辅音 | 例：「がっこう」→ `gakkou` |
| 长音ー | `-`（减号键） | 片假名长音：「サッカー」→ `sakka-` |
| 小假名 | 前面加 `x` 或 `l` | 「ぁ」→ `xa`，「ぅ」→ `xu` |
| 片假名转换 | 输入后按 `F7` | 强制转为片假名 |
| 平假名保持 | 输入后按 `F6` | 强制保持平假名 |
| 半角片假名 | 输入后按 `F8` | 半角カタカナ |

## 三、实际打字流程演示

### 3.1 输入一个完整句子

**目标句子**：わたしは学生です。（我是学生。）

```
步骤1: 输入 watashiha      → わたしは（屏幕显示下划线）
步骤2: 输入 gakusei        → がくせい
步骤3: 按空格键            → 候选列表出现「学生」← 选择
步骤4: 输入 desu           → です
步骤5: 输入 。(句号键)      → 。
步骤6: 按 Enter           → 确认输入
```

> 💡 **关键**：打完一段后按空格键，输入法会自动建议汉字转换。用方向键选择正确的候选词。

### 3.2 常见输入示例

| 目标 | 键入 | 说明 |
|------|------|------|
| こんにちは | `konnnichiha` | 注意「ん」用nn |
| ありがとう | `arigatou` | |
| サッカー | `sakka-` + F7 | 促音+长音+转片假名 |
| 東京 | `toukyou` + 空格选汉字 | |
| 食べます | `tabemasu` + 空格 | |
| ブルーロック | `buru-rokku` + F7 | 外来语用片假名 |

## 四、高效打字技巧

### 4.1 快捷键速查

| 快捷键 | 功能 |
|--------|------|
| `Space` | 汉字变换（选择候选词） |
| `Enter` | 确认当前输入 |
| `Esc` | 取消变换，回到假名 |
| `F6` | 强制转为平假名 |
| `F7` | 强制转为全角片假名 |
| `F8` | 转为半角片假名 |
| `F9` | 转为全角英数 |
| `F10` | 转为半角英数 |
| `Tab` | 展开更多候选词 |

### 4.2 提速建议

1. **先习惯罗马字对应**：最初速度慢是正常的，坚持1-2周会明显提速
2. **多用预测输入**：Windows日语输入法会根据你的打字习惯预测词汇
3. **学会分段转换**：长句子可以分段按空格转换，不需要一次性输入完
4. **善用F7**：输入外来语时先打罗马字再按F7一键转片假名

## 五、练习计划

| 阶段 | 时长 | 练习内容 |
|------|------|----------|
| 第1天 | 10分 | 安装输入法 + 输入あいうえお～わをん |

> 💡 **最佳练习方式**：每天学完新词汇后，用日语输入法把它们打一遍。输入即复习！

## 六、常见问题

**Q: 输入法切换后还是打英文？**
A: 确认任务栏显示「あ」而非「A」。按 `Alt + ~` 切换。

**Q: 「ん」总是变成「な行」？**
A: 记住输入 `nn`。在「ん」后面紧跟元音时尤其重要：「かんあい」→ `kannnai`

**Q: 怎么输入「ー」（长音）？**
A: 按键盘上的 `-`（减号/连字符键）。

**Q: 汉字转换不准确？**
A: 多次使用后输入法会学习你的习惯。也可以分更短的段来转换。
