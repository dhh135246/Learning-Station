## 🎮 Web 学习系统实装特色

本地应用 [index.html](file:///C:/Users/admin/.gemini/antigravity/scratch/japanese-learning-station/index.html) 采用了**深色赛博朋克霓虹美学**设计，支持全离线运行。它已实现以下高级交互模块：

### 1. 🌸 50音道場 (五十音图 + 多功能自测)
- **发音看板**：点击任意假名，即可使用浏览器原生的 **SpeechSynthesis (TTS)** 进行高清发音示范。
- **自测测验**：可自由设定测试范围（平/片/混合）和题型（看假名选罗马字、看罗马字选假名、听音选假名）。答错的假名会记录并显示在“控制中心”的薄弱点归档中。
- **易混淆专项对决**：内置了 `さ vs ち`、`れ vs ね vs わ` 等五组高度相近假名的快速辨识训练。
- **手写临摹沙盒**：内置带有米字辅助线的手写板，支持触屏与鼠标书写。下拉选择任一假名，会在画板背景呈现浅灰字形供你描红临摹。

### 2. 🛠️ 語法工房 (N5 专项进阶)
- **核心语法库**：内置了 9 大 N5 最核心语法点，包括基础助词用法（包含 `は` 与 `が` 辨析）、动词变形（て/た/可能形/たい形）等，配备**动漫经典例句**与 TTS 发音。
- **交互练习**：
  - *助词挖空选择*：高强度判定助词选择。
  - *动词变形挑战*：支持手工键入假名或罗马字验证动词各种变形（同时锻炼日语键入）。
  - *连词成句排序*：将打乱的日文单词碎片通过点击排序，重组为符合语法的日语句子。

### 3. 📖 単語帳 (SM-2 算法卡片)
- **SM-2 间隔重复引擎**：在前端本地实现了完整的 SuperMemo-2 记忆曲线算法。每次翻开单词卡选择熟练度（1-5 阶），算法会动态计算并重排下一次复习到期时间。
- **词汇自测**：支持日译中、中译日及听音选词。
- **数据防丢**：可在控制中心一键将所有生词与算法记录导出为 `.json` 文件，随时随地一键导入恢复。

### 4. 🎵 万能文本解析练习场 (自定义训练轨)
- **歌词/台词导入**：粘贴任何你想练习的日摇歌词、动漫台词。
- **打字伴读练习轨**：系统会自动按行分句，伴随 TTS 逐句发音伴读。用户利用日语输入法在输入框内键入，匹配的字符将自动变绿，回车即可切入下一句，自动统计打字数与吻合率。
- **汉字启发式抓词**：自动提取粘贴文本中的所有汉字生词。用户只需在卡片中补齐假名与中文意思，即可**一键保存至単語帳**开始 SRS 记忆周期。
