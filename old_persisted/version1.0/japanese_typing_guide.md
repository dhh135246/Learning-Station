# 日语 PC 端输入（键入）快速入门指南
# パソコン日本語入力ガイド

对于电脑端学习日语的同学来说，熟练使用日语输入法（IME）是一项必备的底层工具技能。以下为您整理 Windows 系统下日语输入法的安装、转换规则与快捷键指南。

---

## 一、 输入法安装与配置 (IMEのインストール)

### Windows 系统安装步骤：
1. 打开 **设置 (Settings)** -> **时间和语言 (Time & Language)** -> **语言 (Language)**。
2. 点击 **添加语言 (Add a language)**。
3. 在搜索框输入 `Japanese` 或 `日本語`，选择并点击安装。
4. 安装完成后，可通过快捷键 `Win + Space` 在中文输入法和日语输入法之间切换。

---

## 二、 核心转换规则与罗马字对照表 (ローマ字かな変換表)

在日语输入法下，我们使用的是 **罗马字输入法 (Romaji Input)**，即通过键盘敲击对应的英文字母，输入法会自动将其转换为假名。

### 1. 五十音基础行 (五十音清音)
输入最基础的辅音 + 元音即可：

| 行 | あ (a) | い (i) | う (u) | え (e) | お (o) |
|:---:|:---:|:---:|:---:|:---:|:---:|
| **ア行** | a | i | u | e | o |
| **カ行** | ka | ki | ku | ke | ko |
| **サ行** | sa | shi (si) | su | se | so |
| **タ行** | ta | chi (ti) | tsu (tu)| te | to |
| **ナ行** | na | ni | nu | ne | no |
| **ハ行** | ha | hi | fu (hu) | he | ho |
| **マ行** | ma | mi | mu | me | mo |
| **ヤ行** | ya |  | yu |  | yo |
| **ラ行** | ra | ri | ru | re | ro |
| **ワ行** | wa |  |  |  | wo (を) |

> [!TIP]
> - **拨音「ん」**：必须敲击两次 `n`（即 `nn`）。如果只敲击一次 `n`，在后面接元音时会变成「な」行音。
> - **宾格助词「を」**：请键入 `wo`。虽然读音和「お (o)」相同，但在输入法中必须输入 `wo`。

### 2. 浊音与半浊音 (濁音・半濁音)
在清音的基础上，更改辅音字母即可：

| 行 | が (ga) | ざ (za) | だ (da) | ば (ba) | ぱ (pa) |
|:---:|:---:|:---:|:---:|:---:|:---:|
| **辅音** | g | z | d | b | p |
| **示例** | が (ga) | じ (ji/zi) | づ (du) | べ (be) | ぽ (po) |

### 3. 拗音的输入 (拗音 - きゃ、しゅ、ちょ等)
拗音是由“い”段假名与小写的“や、ゆ、よ”拼合而成。
*   **方法一（推荐，直接键入）**：辅音 + `y` + 元音。
    *   `kya` -> きゃ
    *   `shu` / `syu` -> しゅ
    *   `cho` / `tyo` -> ちょ
    *   `rju` / `ryu` -> りゅ
*   **方法二（拆开键入）**：先敲击大写假名，再敲击 `l` 或 `x` + 要变小假名的罗马字。
    *   `ki` (き) + `la` (ぁ) -> きぁ
    *   `ki` (き) + `lyu` (ゅ) -> きゅ （注：`l` 代表 little，`x` 也可以，如 `xyu`）

### 4. 促音与长音 (促音・長音)
*   **促音（小「つ」）**：
    *   **方法（推荐）**：**双写促音后面假名的第一个辅音字母**。
        *   例：*いってきます* (ittekimasu) — 双写 `t`。
        *   例：*やっぱり* (yappari) — 双写 `p`。
    *   **单独输入小「つ」**：输入 `ltu` 或 `xtsu`。
*   **长音（片假名中的「ー」）**：
    *   直接敲击键盘上键盘减号键 `-` 即可。
        *   例：*スローライフ* (suro-raifu) -> `suro-raifu`。

## 三、 高效键入快捷键 (便利なショートカット)

掌握以下快捷键可以免去频繁使用鼠标点击输入法面板的烦恼：

| 快捷键 | 功能说明 |
|---|---|
| **`Alt + ~` (键盘左上角)** | 在 **英文直接输入模式** 与 **日文平假名输入模式** 之间快速切换。 |
| **`Space` (空格键)** | 进行汉字（Kanji）或片假名的转换。多次敲击会展开候选词列表。 |
| **`Enter` (回车键)** | 确认当前输入的文字（消除假名下方的波浪虚线）。 |
| **`F7`** | 将当前输入的罗马字直接转换为 **全角片假名** (例如：`suro-raifu` 转换成 `スローライフ`)。 |
| **`F8`** | 将当前输入的罗马字转换为 **半角片假名**。 |
| **`F9`** | 将当前输入的罗马字转换为 **全角英文**。 |
| **`F10`** | 将当前输入的罗马字转换为 **半角英文**。 |
| **`Esc`** | 清除当前未确认的输入。 |

## 四、 极速练习建议

1.  **初学者练习**：使用本项目配套的 Web 系统的「输入练习 (入力練習)」板块进行动漫名台词输入。
2.  **趣味挑战**：访问日本著名的打字练习网站 [寿司打 (Sushi-da)](http://neutral.x0.com/home/sushida/play.html) 进行日常测验。
3.  **日常习惯**：在电脑端查词典或记录笔记时，强制自己使用日语输入法敲入假名，而非复制粘贴。
