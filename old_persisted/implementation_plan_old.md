# 日语学习系统 — 实施计划

## 项目概述

为一位中文母语、日语初级入门阶段的学习者，构建 **高度定制化的日语学习计划** 和 **配套的 Web 学习应用**。学习者具备50音基础（不够熟练）、约200天多邻国碎片学习经验、大量日漫观看经历，但缺乏系统语法学习。

---

## 产出物总览

| # | 产出物 | 形式 | 说明 |
|---|--------|------|------|
| 1 | 学习计划文档 | Markdown Artifact | 完整的分阶段学习计划，含具体步骤、资源导航、时间分配方案 |
| 2 | 日语键入指南 | Markdown Artifact | Windows 日语输入法配置与使用教程 |
| 3 | Web 学习应用 | 单页 HTML 应用 | 集成多个学习模块的交互式学习平台 |

## 第一部分：学习计划文档

### 计划结构

学习计划将包含以下主要章节：

#### 1. 阶段总览与目标路线图
- **短期（1-3个月）**：巩固50音 → 掌握N5语法核心 → 积累300+基础词汇
- **中期（4-8个月）**：系统完成N5→N4 → 建立基础阅读能力
- **长期（9-18个月）**：N3突破 → N2/N1进阶

#### 2. 核心模块详解（按优先级排序）

| 模块 | 权重 | 说明 |
|------|------|------|
| **语法系统** | ★★★★★ | 最薄弱环节，作为核心重点 |
| **50音巩固** | ★★★★☆ | 短期集中攻克 |
| **词汇积累** | ★★★★☆ | 兴趣导向分类词汇 |
| **听力训练** | ★★★☆☆ | 利用动漫优势强化 |
| **阅读训练** | ★★★☆☆ | 漫画→轻小说渐进式 |
| **日语键入** | ★★★☆☆ | 作为基础工具技能 |

#### 3. 时间分配方案
- **碎片模式（10-30分/天）**：每日打卡任务卡片
- **强化模式（2-4小时）**：深度学习Session设计
- **周末冲刺模式**：模块整合复习

#### 4. 兴趣素材库
按难度分级推荐：
- **异世界/种田系**：《转生史莱姆》《本好きの下剋上》《のんのんびより》等
- **运动系**：《蓝色监狱》《棒球大联盟》《排球少年》等
- **日摇推荐**：适合学习的乐队/歌曲（歌词学习法）
- **漫画素材**：适合各阶段阅读的漫画推荐

#### 5. 语法专项攻略
- N5语法核心30条详解（日中双语）
- 语法学习三步法：理解→例句→实战
- 动漫常见语法句型归纳
- 外部资源导航（教材、网站、App推荐）

#### 6. 日语键入完整指南
- Windows 日语输入法安装与配置
- 罗马字输入法基础对照表
- 常用键入技巧与快捷键
- 练习方法与推荐练习平台

## 第二部分：Web 学习应用

### 应用功能模块

```
┌─────────────────────────────────────────────┐
│              日语学習ステーション               │
│          Japanese Learning Station            │
├─────────────────────────────────────────────┤
│                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │ 50音道場 │  │ 語法工房 │  │ 単語帳   │  │
│  │ 假名练习 │  │ 语法学习 │  │ 词汇卡片 │  │
│  └──────────┘  └──────────┘  └──────────┘  │
│                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │ 聴力特訓 │  │ 入力練習 │  │ 学習記録 │  │
│  │ 听力训练 │  │ 打字练习 │  │ 进度追踪 │  │
│  └──────────┘  └──────────┘  └──────────┘  │
│                                             │
└─────────────────────────────────────────────┘
```

#### 模块1：50音道場（假名练习）
- 平假名/片假名识别测验
- 听音选假名
- 手写（点击）练习
- 易混淆假名专项
- 进度追踪与薄弱点分析

#### 模块2：語法工房（语法学习）
- N5/N4语法卡片系统
- 语法点详解（日中双语释义 + 例句）
- 填空/选择/造句练习
- 动漫例句关联
- 语法分类浏览（助词/动词变形/句型）

#### 模块3：単語帳（词汇系统）
- 分主题词汇库（异世界、运动、日常）
- 闪卡翻转学习
- 间隔重复（SRS）算法
- 词汇测验（日→中 / 中→日 / 听音选词）

#### 模块4：聴力特訓（听力训练）
- 常用短语听辨
- 动漫经典台词听写
- 听力理解小测验

#### 模块5：入力練習（打字练习）
- 罗马字→假名输入练习
- 单词输入练习
- 句子输入练习
- 速度与准确率统计

#### 模块6：学習記録（进度追踪）
- 每日学习时长记录
- 各模块完成度可视化
- 连续学习天数
- 成就系统

### 技术方案

| 项目 | 选择 | 理由 |
|------|------|------|
| 框架 | 纯 HTML + CSS + JavaScript | 单文件部署，无需构建工具 |
| 样式 | Vanilla CSS + CSS Variables | 深色主题，动漫风格美学 |
| 数据存储 | localStorage | 离线可用，无需后端 |
| 字体 | Noto Sans JP + Inter | 中日文完美支持 |
| 动画 | CSS Animations + JS | 流畅的界面交互 |

### 设计风格
- **深色主题**为主，搭配霓虹色调（赛博朋克/动漫风格）
- **玻璃拟态**卡片设计
- 日式元素装饰（樱花、波纹等微动画）
- 响应式布局，适配不同屏幕
