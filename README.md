# 学習局Learning-Station — 快速上手指南

学習局Learning-Station是一款专为中文日语学习者定制的**极客风、极致日式和风 (Wafuu Modern style)** 离线学习与打字对战平台。本系统完全基于纯前端技术栈构建，不依赖任何云端数据库与后台服务，100% 保护您的个人数据隐私。
直接访问地址：https://dhh135246.github.io/Learning-Station/

---

## 🌟 核心特色功能

1. **⌨️ 罗马字 ➔ 假名实时转换输入引擎**
   - 聚焦文本框直接键入罗马字（如 `sushida`），输入框将实时、流畅地变为日语假名（`すしだ`）。
   - 支持快捷键 `Alt + Q` 或 `Ctrl + M` 进行“平假名 (あ)/片假名 (ア)/英文 (A)”三模无缝循环切换，并带有清脆的和风电子音效反馈。
   - 内置光标位置防跳跃保护，即便在长句子中段插入修改，光标也绝不乱跳。
2. **📅 科学抗遗忘 SRS (SM-2) 智能单词本**
   - 单词复习卡片支持 3D 立体中轴旋转 (rotateY) 交互。
   - 结合每日到期单词量，自动渲染未来 7 天待复习日历看板。根据待复习单词多寡，自动呈现五级记忆衰退着色波动，方便规划学习。
   - **可配置入队限额**：您可在控制中心自主修改每日新词导入或积压新词平摊的每日上限（5~100个/天），系统会自动完成平滑的天级顺延排程。
3. **🎮 次元道场趣味学习中心**
   - **文本解析轨**：复制任意日文歌词或文章，系统会基于前缀 Trie 树将文本智能双语切分为生词，自动与单词本碰撞并预填释义。您可伴随打字音效和匹配度高亮框进行打字跟读挑战。
   - **异世界冒险 (JRPG)**：通过假名拼写和词义选择，扮演勇者击败魔物！包含普通小怪与关底 Boss 赤鬼 `👹` 决战、无伤过关奖励“生命药水 🧪”、以及纯本地 Web Audio 合成的 8-bit 复古对战音效。
   - **名台词连连看与体育外来语配对**：动漫名台词和足球/棒球外来语的假名/释义配对卡牌对决。
4. **🔍 免跨域 Weblio (中日) 随身辞书与避让式抽屉**
   - 彻底解决查词 502 痛点，接入权威 Weblio (中日) 在线词典。
   - 点击屏幕右下角 `🔍` 挂坠，可在右侧拉出 `360px` 的随身辞书抽屉。
   - 展开时，主界面内容会自动平滑向左平移收缩避让，**100% 杜绝挡住打字框或视线**。背景无遮罩，支持边看释义边在一旁打字，敲击回车直接一键查词。
5. **📓 私人 Markdown 语法笔记导入**
   - 支持拖拽本地 `.md` 文件或粘贴 Markdown 源码同步存盘。
   - 本地自研 RegExp 轻量渲染器支持多级标题、粗斜体、GitHub alert 风格警告气泡、以及带 `🔊` 的点击朗读发音链接。
6. **🌸 禅意和风昼夜美学**
   - 漫天樱花缓缓飘落的 Canvas 背景动效，在“夜樱模式”下花瓣呈紫色发光，且切换模式时播放清脆的敲竹声（鹿威击石音效）。

## 🚀 快速启动与安装指南 (PC/手机端)

### 1. 本地双击直接运行 (零门槛)
本系统完美兼容 `file:///` 协议。无需部署任何服务器，直接在电脑的文件管理器中双击 [index.html](file:///f:/Japanese/customised_system/index.html)，即可在浏览器中本地启动并流畅使用所有功能。

### 2. 部署并安装为桌面独立 App (推荐)
如果您想获得更完美的无边框沉浸式窗口体验，可以将系统部署为 **PWA (渐进式 Web 应用)**：
1. **启动轻量级静态服务**：在项目根文件夹下打开终端运行：
   ```bash
   npx serve -l 8080
   # 或者使用 Python：python -m http.server 8080
   ```
2. **访问并安装**：
   * 打开 Chrome/Edge 浏览器访问 `http://localhost:8080`；
   * 观察浏览器地址栏右侧，会浮现一个“电脑屏幕+下箭头”的安装图标，点击确认；
   * 系统将自动在您的桌面上生成快捷 App 窗口。离线后打开此 App，由于 Service Worker（`sw.js`）的离线拦截，系统依然能够零延迟完美加载！

## 🔒 隐私与多设备数据同步

本系统将您的假名进度、SRS 生词本、JRPG 排行榜、以及私人语法笔记 **100% 保存在您当前设备的浏览器 LocalStorage 中**。由于浏览器的同源安全沙盒政策，双击运行版与 PWA 安装版的数据是不互通的。

**数据同步与备份闭环**：
您可以在旧设备或 file 协议版系统的 **「控制中心」 ➔ 点击「导出备份数据 (JSON)」** 导出进度包；然后在新设备或 PWA 版 App 中，将该 `.json` 备份文件**直接拖入「备份与控制中心」的 📁 拖放盘中**，系统即会秒级反演，完美迁移合并您的全部学习资产！
